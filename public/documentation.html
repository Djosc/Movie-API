My Flix Documentation

Action Request URL Response
Return all movies GET /movies Returns JSON data about all movie objects
Returns data about a single movie GET /movies/:title Return JSON data about a single movie based on the title passed in the URL
Return a genre description GET /genres/:genre Returns a description about a genre based on the genre passed in the URL
Returns data about a director GET /directors/:director Return JSON data about a director based on the director passed in the URL
Registers a new user POST /users Returns a success (201) or failed (400) status code for the POST request
Updates a user's username PUT /users/:userName Returns a success (201) or failed (400) status code for the PUT request
Adds a movie to a list of favorites POST eg. { title: 'The Shawshank Redemption', director: 'Frank Darabont', genre: ['Drama'], releaseYear: '1994' } /users/add/:movie Returns a text response confirming the movie was added successfully
Removes a movie from a list of favorites DELETE /users/remove/:movie Returns a text response confirming the movie was removed successfully
Deletes the user's account DELETE /users/deleteAccount Returns a text response confirming the user account was successfully deleted
