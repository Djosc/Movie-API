My Flix Documentation

Action Request URL Response
Return all movies GET /movies Returns JSON data about all movie objects.
Returns data about a single movie by Title GET /movies/:Title Return JSON data about a single movie based on the title passed in the URL. eg:

{

Title: String,

Description: String,

Genre: { Name: String, Description: String },

Director: { Name: String, Bio: String, Birth: Date, Death: Date },

Actors: [String],

ImagePath: String,

Featured: Boolean

}
Return a genre description GET /genres/:Genre Returns a description about a genre based on the genre name passed in the URL
Returns data about a director by Name GET /directors/:Director Return JSON data about a director based on the director passed in the URL. eg:

{

Name: String,

Bio: String,

Birth: Date,

Death: Date

}
Returns data all users GET /users Returns JSON data about all users.
Returns information about a user by Username GET /users/:Username Returns JSON data about a user. eg:

{

Username: String,

Password: String,

Email: String,

Birthday: Date,

FavoriteMovies: [String]

}
Registers a new user POST eg. JSON body:

{

Username: String,

Password: String,

Email: String,

Birthday: Date

}

 /users If successful, returns a JSON object that matches the data passed in the body.
Updates a user's info by username PUT eg. JSON body:

{

Username: String,

Password: String,

Email: String,

Birthday: Date

}

 /users/:Username If successful, returns a JSON object that matches the data passed in the body.
Adds a movie to a user's list of favorites POST /users/:Username/movies/:MovieID If successful, returns a user's updated JSON data. eg:

{

Username: String,

Password: String,

Email: String,

Birthday: Date,

FavoriteMovies: [String]

}
Removes a movie from a user's list of favorites DELETE /users/:Username/movies/:MovieID If successful, returns a user's updated JSON data. eg:

{

Username: String,

Password: String,

Email: String,

Birthday: Date,

FavoriteMovies: [String]

}
Deletes a user's account by Username DELETE /users/:Username On success, returns confirmation of deletion. On fail, returns a 400 status code.
